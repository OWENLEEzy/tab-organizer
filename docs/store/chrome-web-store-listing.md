# Chrome Web Store Listing

## Name
Tab Out

## Short Description
Clean up tab overload with a local dashboard grouped by product and domain.

## Detailed Description
Tab Out turns your open Chrome tabs into a focused dashboard. Click the toolbar icon to see tabs grouped by product or domain, spot duplicates, close clutter, organize product groups into sections, and recover recent local sessions after a restart.

Core features:
- Automatic product/domain grouping for Gmail, GitHub, Vercel, YouTube, Google Docs, Drive, Calendar, and more.
- Manual sections for organizing product groups without losing automatic grouping.
- Cards and Table views for quick scanning.
- Duplicate detection and one-click duplicate cleanup.
- Recent session recovery from local snapshots with preview-before-restore.
- 100% local storage. No account, no server, no external sync.

## Privacy Copy
Tab Out stores tab URLs, titles, favicons, product/domain metadata, manual sections, and recent recovery snapshots in `chrome.storage.local` on your device. It does not send this data to any server and does not require an account.

Recovery snapshots are capped to the latest 5 sessions and 80 real web tabs per snapshot. Browser-internal pages, extension pages, and Tab Out pages are excluded.
