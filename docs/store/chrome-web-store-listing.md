# Chrome Web Store Listing

## Name
Tab Organizer

## Short Description
Clean up tab overload with a local dashboard organized into groups.

## Detailed Description
Tab Organizer turns your open Chrome tabs into a focused dashboard. Click the toolbar icon to see tab groups, spot duplicates, close clutter, organize groups into sections, and recover recent local sessions after a restart.

Core features:
- Automatic grouping for Gmail, GitHub, Vercel, YouTube, Google Docs, Drive, Calendar, and more.
- Manual sections for organizing groups without losing automatic grouping.
- Cards and Table views for quick scanning.
- Duplicate detection and one-click duplicate cleanup.
- Recent session recovery from local snapshots with preview-before-restore.
- 100% local storage. No account, no server, no external sync.

## Privacy Copy
Tab Organizer stores tab URLs, titles, favicons, product/domain metadata, manual sections, and recent recovery snapshots in `chrome.storage.local` on your device. It does not send this data to any server and does not require an account. The extension requests `tabs`, `storage`, and `favicon` permissions to display tab groups and their icons.

Recovery snapshots are capped to the latest 5 sessions and 80 real web tabs per snapshot. Browser-internal pages, extension pages, and Tab Organizer pages are excluded.
